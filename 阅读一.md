### 审题

- 明确字数
- 看标题明确主题
- 出现指代词this/that/these/those/such等，答案会从一个句子转移到另一个句子(一般往前面找)
- 注意数字，不单是阿拉伯数字



> Summary 看标题和summary段标题一不一样，不一样就去定位段落；然后再去找找定位词什么的。



### 同义替换

1. 词性转换
2. 同义词、近义词
3. 上下义词（即抽象到具体）
4. 反义词的否定
5. 语态转换（主被动的转换）



### 图表填空

> summary填空也是 顺序原则，偶有乱序

- 答案相对集中
- 顺序原则，偶有乱序
- 原文原词



### 判断题

#### 比较级

1. 先看比较双方是否存在（定位词）不存在NG
2. 看比较关系是否存在（考点）不存在NG
3. 前两点都存在的话，看比较关系是否一致（一致-T..）





### 匹配题

#### 特殊词匹配

##### 特点

- 细节题

- 出题范围：全文局部都有

- 顺序原则：乱序

- 是否可能出现复选(NB)：是

> 特殊词包括**人名、国家、国籍、城市、时间**等

#### 解题步骤
1. 审题：明确题目要求(关注是否有NB)
2. 理解题目 划出重点单词和短语
3. 定位：在原文中找出特殊词所所在段落
4. 扫读出题段落 在特殊定位词附近找出题干同义替换 明确答案


#### 分类匹配
##### 特点

- 能否找到原词：是

- 顺序原则：乱序

- 细节题 都是句子

#### 解题步骤
1. 审题：看选项有无重合
  - 无重合，直接用选项定位
  - 有重合，用重合的部分定位
2. 理解题目 划出重点单词和短语
3. 定位：在原文中找出特殊词所所在段落
4. 扫读出题段落 在特殊定位词附近找出题干同义替换 明确答案

### 简答题
文章细节信息；二者皆有；顺序

### 小标题(LOH)
#### 特点
- 出题目的：辨别核心内容&重要细节信息
- 出题范围：指定段落 和 全文(居多) 都有
- 题目位置：文章前

#### 解题步骤
1. 划出标题中的关键词
2. 阅读出题段落，确定段落核心内容
  - 有主旨句则看**主旨句**（首/二 + 末句、转折词之后）
  - 若无主旨句则扫读全段
3. 选择与段落**核心内容**有**同义替换**关系的标题选项
4. 如果无核心内容的替换，选择与**重要细节**信息有**替换**的选项

> 段中找两点
> - 关键词一个：会重复出现
> - 关键词两个：找到各自的替换





> 补充：Category - kind - type - variety - classification 都是分类的意思



### 段落信息匹配

#### 特点

- **全文**出题，**乱序**
- 细节题-用**句子**出题
- 题干中有“contain” 常有NB提示有复选项（小概率出现有NB但无复选，例C13T3P3）
- 定位词主要以**替换**形式出现，需要理解题意来确定

#### 解题步骤

1. 找和其他已完成的题目中是否有同类信息或单词替换
2. 有**特殊关键词**如数字/人名/术语/时间地点等的可以优先做
3. 完整阅读并理解题意，划出定位词后回文章定位

> 注意：难找题目最后做，优先在未出题（或出题少）的段落中选



#### 特殊关键词

- 含有rate / ratio / proportion / percentage 等词，往往对应**%**的段落（例C9P43Q6）
- 含有cost / fund / financial / business / income / salary / wage / commercial / dealing / purchase / sale / expense / pension等词，往往对应**金钱符号**的段落（例如C7P46Q14）
- 含有time / period / age等词，往往对应**时间**的段落（例如C9P93Q16）
- 含有development / progress / process / stage / procedure / phase等词，往往对应**表达次序或者时间顺序**的段落（例C7P46Q15）
- 含有introduction  / orientation / definition / overview等词，一般对应**开头**的段落（例如C6P20Q6）
- 含有effect / impact / prospect / future / suggestion / prediction / conclusion / summary等词，一般对应**结尾**段落（例C9P47Q16）
- 含有location / places等词，一般对应**地点**的段落（例C9P69Q14）



### 句首配句尾

#### 题型特点

- 基本有序，局部乱序
- 出题范围：均有可能

#### 解题步骤

1. 翻译题干，划出题干中的定位词
2. 回原文定位，找到答案句
3. 精读答案句，划出重点内容
4. 在选项中找出重点内容的同义替换，确定答案



### 选择题（单选&多选）

#### 单选题 

- 细节题
- 主旨大意题（放在最后做！）

##### 题型特点

- 1段可出1-3题
- 基本有序
- 细节题看句子；主旨大意题看段落概括（通常是首尾两段）

##### 解题步骤

1. 审题，明确出题段落
2. 翻译题干，划出题干中定位词
3. 回到原文找到答案句，确定核心内容
4. 在选项中找出核心内容的**同义替换**，确定答案

#### 多选题

##### 题型特点

- 答案来源：句子
- 全文或者某段或连续某两段
- 所选项正确，但顺序与答案不一致，不影响得分

##### 解题步骤

1. 审题，明确答案数量要求以及题干定位词（题干没定位词就去看选项，然后去定位）
2. 定位出题位置
3. 阅读选项，划出选项关键词
4. 比对原文与选项内容，排除错误内容，选择正确内容
